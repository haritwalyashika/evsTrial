We aim to solve the demerits of offline voting system by providing an online voting system which can overcome the barriers of the electoral voting system. Salient Features:

  • Platform agnostic. Can be operated from Desktop, laptop, mobile as well as any browser on any operating system.
  • Protection from voting multiple times.
  • Audit trail to log activities carried out by Voter as well as candidate

Steps to perform if you are Voter:

  1. Register yourself as voter, if not registered so far.
  2. Login as voter
  3. Select the election for which you want to vote
  4. Cast your vote
  5. Check Election Result

Steps to perform if you are Candidate:

  1. Register yourself as Candidate, if not registered so far.
  2. Login as Candidate
  3. Update your profile including past performances and Manifesto
  4. Check Election Result
